    6. Split a String

  • You can split strings based on what separates them and than being able to access it by its list value.
  • The example below splits a string based on if it can be split by spaces.
    • string = "My name is Jon and i live in a button factory"; split = string.split(); print(split)
  • The example below splits a string based on if there are dashes in the string.
    • string = "My-name-is-Jon-and-i-live-in-a-button-factory"; split = string.split('-'); print(split)
  • Notice that when using the split command, it outputs it to an Array / list "has blocky quotes in it". You can access a value by specifying the numbered index in the list. In programming, the first element in an index is usually zero while the last element is indexed by -1
    • print(split); print("Access the 1st Attribute"); print(split[0]); print("Access the Last Attribute"); print(split[-1])
  • if you want to make the output prettier you can use the + syntax and str() commands to combine them
    • print(split); print("The 1st Attribute is: " + str(split[0])); print("The Last Attribute is: " + string(split[-1]) );
