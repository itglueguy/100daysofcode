    2. Add an item to a List and print it out

  • We can use the .append() command on the list object to add an item to our list
    • dog_breeds = ["daschund","chihuahua","Golden Retriever"]; print(dog_breeds); dog_breeds.append("Corgi"); print(dog_breeds)
  • Lets see what happens when we add a duplicate item - will that matter?
    • dog_breeds = ["daschund","chihuahua","Golden Retriever"]; print(dog_breeds); dog_breeds.append("Corgi"); dog_breeds.append("Corgi"); print(dog_breeds)

    3. Remove an item from a List and print it out

  • We can use the .remove() command on the list object to add an item to our list
    • dog_breeds = ["daschund","chihuahua","Golden Retriever"]; print(dog_breeds); dog_breeds.remove("daschund"); print(dog_breeds)
  • Lets see if it will remove duplicate items from our list
    • dog_breeds = ["daschund","chihuahua","Golden Retriever","daschund"]; print(dog_breeds); dog_breeds.remove("daschund"); print(dog_breeds)
  • interesting. Even though there are duplicated items; it only removes 1 item at a time.

    4. Add Items to a list from another list

  • You can initiate a list by equating a variable name to an empty bracket list by using the format: my_list_name = []
    • my_list_with_items = ['pens','paper','pencils']; my_empty_list = []; print (my_list_with_items); print(my_empty_list);
  • lets see if we can use the .append() command to copy things from one list to another
    • # The list with items in it my_list_with_items = ['pens','paper','pencils']; # The list with no items in it my_empty_list = []; # lets append the first item of the array my_empty_list.append(my_list_with_items[0]); print ("Items in my empty list:") print(my_empty_list); # After we print it again, it will now show in my list again print ("Items in my empty list:") my_empty_list.append(my_list_with_items[1]); print(my_empty_list);
  • Let me get some practice in by using a For loop to copy the items over
    • # The list with items in it my_list_with_items = ['pens','paper','pencils']; # The list with no items in it my_empty_list = []; # Github copilot really does help cheat on the learning of python. for item in my_list_with_items: my_empty_list.append(item); print(my_empty_list);
