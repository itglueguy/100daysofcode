    5. Dealing with Dictionaries ( hashtables )

  • Dictionaries or hashtables seem to have multiple Keys in which each key has 1 value
  • In the example below, the Key of Name is equal to Ron, their Age is 20, and their hobby is 3d printing
    • dictionary_hashtable = {'Name': 'Ron', 'Age': 20, 'Hobby': '3d Printing'}; print(dictionary_hashtable);
  • In order to access a value in the dictionary, we need to specify the key in order to get the value in question
    • dictionary_hashtable = {'Name': 'Ron', 'Age': 20, 'Hobby': '3d Printing'}; name = dictionary_hashtable['Name']; print(name); age = dictionary_hashtable['Age']; print(age);
  • Let see if we can add a new key to our existing dictionary
    • dictionary_hashtable = {'Name': 'Ron', 'Age': 20, 'Hobby': '3d Printing'}; print(dictionary_hashtable); dictionary_hashtable['Profession'] = 'Ninja'; print(dictionary_hashtable)

    6. Modifying dictionary values

  • We can modify a key in a dictionary simply by specifying its key and equating it to its new value
    • dictionary_hashtable = {'Name': 'Ron', 'Age': 20, 'Hobby': '3d Printing'}; print(dictionary_hashtable); dictionary_hashtable['Hobby'] = 'Programming'; print(dictionary_hashtable);
  • Lets see if we can use the .clear() method on the dictionary to delete one of the keys and its value
    • dictionary_hashtable = {'Name': 'Ron', 'Age': 20, 'Hobby': '3d Printing'}; print(dictionary_hashtable); del dictionary_hashtable['Age']; print(dictionary_hashtable);
